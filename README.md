# Darth Buddha's - Trading View Scripts

## Scripts

* Buddha Bands - Four SMA bands 20-21, 50-55, 200-233 and 365-377.
* Buddha Squeeze - TTM Squeeze style oscillator.
* Market Levels - Auto draws key market levels.
* Market Sessions - Helps keep track of daily market session.
* Price Line - Helps display current price.
* Volume Delta - Buy vs Sell volume.
* Volume Delta Panel - Buy vs Sell volume panel.
